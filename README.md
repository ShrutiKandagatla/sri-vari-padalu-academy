Sri Vari Padalu Academy
Description
Sri Vari Padalu Academy is a full-stack web application repository maintained by Shruti Kandagatla. While the repository is in its early stages following its initial commit, it is structured to support both server-side and client-side development.
Project Structure
The repository contains the following main components:
backend/: This directory contains the server-side code and API logic.
frontend/: This directory contains the user interface and client-side code.
requirements.txt: This file lists the Python dependencies necessary to run the backend environment.
